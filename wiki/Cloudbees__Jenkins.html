Cloudbees & Jenkins¶

Cloudbees¶

We have choosen Cloudbees as a production solution for our solution.

Continuous Integration¶

This project is deployed with Jenkins, an open source continuous integration tool.
Jenkins link : https://drusy.ci.cloudbees.com/job/grailsoverflow/
Deployed app link : http://grow.drusy.cloudbees.net (BE PATIENT, IT COULD BE SLOW)

The process of the CI job is simple :
  • Compute some changes on the grailsw deploy files
  • Change line separators from dos to unix (using tr command)
  • Make it executable (using chmod +x)
  • Build the app
  • Test the app
  • Deploy the app
