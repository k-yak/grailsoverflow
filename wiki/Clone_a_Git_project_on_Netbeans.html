Clone a Git project on Netbeans¶

Pre-requisites :
Install Netbeans
Install groovy
Install grays (for windows no blankspace in the GRAYS_HOME path)

WARNING : The grails version must be 2.2.1 to fit the continuous integration on CloudBees

Process :
  1. Install grails plugin on Netbeans IDE
  2. Choose Team > Git > Clone from the main menu
  3. Insert: https://github.com/loof42/grailsoverflow.git your login, pass and click next
  4. Check the master checkbox
  5. Accept to create a project in Netbeans
  6. Enjoy
